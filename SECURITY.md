# Security Policy

## Reporting Issues

If you find a security or privacy issue, please open a GitHub issue with a minimal description that does not expose secrets or private business data.

Do not include:

- Tokens, cookies, sessions, or account credentials.
- Supplier quotes, customer records, order data, or advertising reports.
- Raw ASIN exports that contain private annotations.
- Internal documents, screenshots, or local file paths.

## Data Handling

This repository is intended to contain only public documentation, generic prompt templates, and anonymized examples. It should not contain operational data or private Amazon seller records.

Before publishing changes, run a quick local scan for:

- `.env`
- `cookie`
- `session`
- `token`
- `secret`
- `.xlsx`
- `.pptx`
- `.pdf`
- raw JSON exports

## Scope

This project provides research workflow guidance. It does not provide legal, compliance, patent, financial, sourcing, customs, or marketplace-policy advice.
