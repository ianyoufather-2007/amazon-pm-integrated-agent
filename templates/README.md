# Stage-Gate Prompt Templates

These templates turn the 000-007 workflow into copy-ready prompts.

Use them when you already know which stage you want to run. If the stage is unclear, start with `stage-000-triage.md`.

## Template Index

| Stage | Template | Use When |
| --- | --- | --- |
| 000 | `stage-000-triage.md` | Only a loose product idea or category direction is available |
| 001 | `stage-001-company-fit.md` | You need to check whether the opportunity fits the team's constraints |
| 002 | `stage-002-demand-trend.md` | You have keyword, BSR, trend, or demand signals |
| 003 | `stage-003-market-structure.md` | You have BSR/category data and need to understand competition structure |
| 004 | `stage-004-competitor-map.md` | You have competitor ASINs and need direct competitor mapping |
| 005 | `stage-005-voc-definition.md` | You have reviews/Q&A/listing evidence and need VOC/product definition hypotheses |
| 006 | `stage-006-roi-risk.md` | You have cost, FBA, quote, compliance, patent, or sourcing assumptions |
| 007 | `stage-007-validation.md` | You have samples, QC notes, test results, or real-world validation evidence |

## Shared Rules

- Do not invent sales, search volume, profit, compliance, patent, or review conclusions.
- Mark missing evidence as P0 / P1 / P2.
- Separate facts, interpretation, risks, gaps, and decisions.
- Do not use final Go/Kill language unless the user says a meeting has already made that decision.
- Default marketplace is Amazon US unless the user says otherwise.
