# Stage Gates

## Standard Flow

```text
000 快筛
001 公司匹配
002 趋势需求
003 市场结构
004 竞品 ASIN
005 VOC 产品定义
006 ROI / 供应链 / 合规 / 专利
007 实物验证
Excel / PPT / QA
```

## Round 1: 市场宽筛

输入：

- BSR Top100/200/300/400
- 主关键词
- 可选 ABA / CPC / Google Trends

允许输出：

- 市场判读
- 候选主攻形态
- 竞品 ASIN 初选
- P0/P1/P2 下载清单

禁止输出：

- 最终产品定义
- 最终 ROI
- 量产建议

## Round 2: 竞品深拆

输入：

- 反查关键词
- Review / Q&A
- Keepa
- Listing 截图或页面信息

允许输出：

- 竞品功能矩阵
- VOC22
- FACT
- JTBD
- 产品定义假设
- 样品测试清单

禁止输出：

- 最终 ROI 确认
- 最终 Go / No-Go
- 量产建议
- 实物验证结论

## Round 3: 供应链和实物验证

输入：

- 报价单
- BOM
- FBA / 头程 / 关税
- 样品 / QC
- 合规认证
- 专利 / FTO
- 竞品实物拆测

允许输出：

- ROI 区间
- 供应链验证
- 决策支持包
- 实物验证报告
- 不可逆风险登记

禁止输出：

- 在证据不足时强行裁定
- 报告层直接写“建议切入 / 不建议切入”
